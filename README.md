# Prism Equity Partners - Landing Page

A premium, high-converting landing page for Prism Equity Partners - a boutique tax strategy and wealth-building firm.

## Features

- ✨ **Dark Mode Premium Design** - Sleek, modern, and high-end aesthetic
- 🎨 **Dynamic Animations** - Smooth scroll interactions and fade-in effects
- 📱 **Fully Responsive** - Mobile-first design that works on all devices
- ⚡ **Performance Optimized** - Built with Next.js for fast loading
- ♿ **Accessible** - Semantic HTML and keyboard navigation support
- 🎯 **Conversion Focused** - Multiple CTAs and email capture forms
- 🌈 **Emerald & Gold Accents** - Premium color palette representing wealth

## Tech Stack

- **Framework**: Next.js 16+
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Hosting**: Vercel

## Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the site.

### Production Build

```bash
npm run build
npm start
```

## Project Structure

```
prism/
├── app/
│   ├── layout.tsx      # Root layout
│   ├── page.tsx        # Main page
│   └── globals.css     # Global styles
├── components/
│   ├── Header.tsx      # Navigation header
│   ├── Hero.tsx        # Hero section
│   ├── ProblemSection.tsx     # Problem/pain point
│   ├── SolutionSection.tsx    # Solution/framework
│   ├── CTASection.tsx   # Call-to-action
│   └── Footer.tsx      # Footer
├── tailwind.config.js  # Tailwind configuration
└── tsconfig.json       # TypeScript configuration
```

## Sections

1. **Header** - Sticky navigation with CTA button
2. **Hero** - Main headline, subheadline, and dual CTAs with animated prism shape
3. **Problem** - Pain points and statistics
4. **Solution** - The "Prism Framework" with 3-column grid
5. **CTA** - Email signup and additional trust signals
6. **Footer** - Links and legal disclaimers
